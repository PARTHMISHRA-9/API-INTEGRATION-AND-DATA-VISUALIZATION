# Weather API Data Visualization

## Description
This project fetches real-time weather forecast data from the OpenWeatherMap API
and visualizes temperature trends using Python and Matplotlib.

## Technologies Used
- Python
- OpenWeatherMap API
- Requests
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your OpenWeatherMap API key in `main.py`

3. Run the script:
   python main.py

## Output
The script displays a line graph showing temperature forecast for the selected city.
